# Facial-Emotion-Recognition-using-Deep-Learning
Deep Learning Approach to Facial Expression Recognition (FER-2013)This project develops a Deep Learning model using a Convolutional Neural Network (CNN) to accurately classify human facial expressions into seven distinct emotion categories: Angry, Disgust, Fear, Happy, Sad, Surprise, and Neutral. Beyond general sentiment analysis, this robust model is designed to power high-value systems such as Smart Classroom Assistants for monitoring student engagement, Driver Safety Systems for detecting drowsiness, and Telehealth Tools for quantifying patient emotional trends.Authors: Kalvala Karunakar; Chinya Umesha, Pranav Karthik; Bharath Sampekatte (Team 2) Course: ASDS-5305 Deep Learning/Neural Networks 📊 Dataset OverviewThe model utilizes the FER-2013 benchmark dataset from Kaggle.Format: 48x48 pixel grayscale images.Data Volume: 28,709 training images and 3,589 images each for testing and validation.The Challenge: The dataset features severe class imbalance, with the majority class ("Happy") containing 7,215 images and the minority class ("Disgust") containing only 436 images. The images also present high variability in lighting, head pose, and occlusions, alongside low-resolution constraints.⚙️ Data Preprocessing & PipelineTo ensure the model learns robust geometric features rather than relying on pixel clarity, several preprocessing steps were implemented:Normalization: Pixel intensity values were scaled from [0, 255] down to a strict [0.0, 1.0] range.Data Augmentation: Applied Random Flip, Random Rotation (10%), and Random Zoom (10%) to artificially expand the training set and improve generalization.🧠 Model ArchitectureThe network is built on a Custom Sequential CNN containing 4 distinct feature extraction blocks:Block 1: 32 Filters for initial feature extraction.Block 2: 64 Filters.Block 3: 128 Filters.Block 4: 256 Filters to capture high-level abstract features and complex facial micro-expressions.Regularization: A Dropout(0.25) layer is applied after every block to prevent overfitting.Classifier: A Dense Layer of 512 units feeding into a 7-class Softmax activation function.🛠️ Optimization StrategiesTo overcome the inherent biases of the dataset, specialized optimization techniques were utilized:Dampened Class Weights: Used the Square Root method to handle severe class imbalance. This reduced the raw inverse frequency weight of the "Disgust" class from 9.41 down to 3.07, preventing the model from over-predicting the minority class.Dynamic Learning Rate: Implemented ReduceLROnPlateau, which halves the learning rate if the validation loss fails to improve for 3 consecutive epochs, allowing the model to settle safely into the global minimum.📈 Results & PerformanceThe customized CNN successfully aligns with human-level performance benchmarks on this specific dataset.Overall Test Accuracy: 65.51% (improved from a 50% baseline).Key F1-Scores: Happy achieved 87%, Surprise reached 75%, and Disgust hit 56% (a significant achievement given it is the minority class)
